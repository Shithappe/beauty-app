## How to run

- `composer install` and `npm install` to install project
- change `.env` (connect to  DB)
- `php artisan serve` - start backend. `npm run dev` - start frontend

### Power by

- Backend: Laravel + Inertia
- Frontend: Vue
- Database: MySQL
